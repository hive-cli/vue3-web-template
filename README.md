# vue3 web 端基础模版

## 安装运行

npm 安装

```bash
npm i # 安装依赖
npm run dev # 运行
```

pnpm `推荐`

``` bash
pnpm i # 安装依赖
pnpm run dev # 运行
```

## 已整合功能

- naive-ui 组件库
- pinia 状态管理
- 路由
- windicss CSS 库
- 请求封装、api 分离、错误自动提示
- 自动化规范代码格式（需自行安装 prettier 插件）
- 跨域配置
- 组件自动导入
- 图标自动化按需使用
- 组件库主题修改
- echarts 图表组件
